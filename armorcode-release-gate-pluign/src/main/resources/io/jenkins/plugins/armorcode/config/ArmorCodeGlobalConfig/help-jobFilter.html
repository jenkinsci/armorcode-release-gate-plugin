Controls which jobs are monitored by ArmorCode:

  • Uses regular expressions for pattern matching
  • Separate multiple patterns with commas
  • Prefix with ! to exclude matching jobs

Examples:

  • .*production.* - Monitor only jobs with "production" in name
  • !.*test.* - Exclude jobs with "test" in name
  • prod-.*, staging-.*, !.*-test - Include jobs starting with "prod-" or "staging-", exclude jobs ending with "-test"
