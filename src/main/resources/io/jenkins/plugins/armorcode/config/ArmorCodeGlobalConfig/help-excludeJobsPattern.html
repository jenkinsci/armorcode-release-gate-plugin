Controls which jobs are excluded from monitoring by ArmorCode:

  • Uses regular expressions for pattern matching.
  • Separate multiple patterns with commas.
  • If a job matches both an include and an exclude pattern, it will be excluded.

Examples:

  • .*test.* - Exclude jobs with "test" in the name.
  • dev-.*,staging-.* - Exclude jobs starting with "dev-" or "staging-".
